### 開發環境
- Vuforia 9.3.3
- Unity 2019.4.6f1(LTS)

---
### 專案包含
此專案純粹為練習Unity AR的操作.<br>
物件包含了川普、狗及手機搖桿.

----
### 專案使用
將vuforia license匯入包內，切換成Android平台，即可build。<br>
將相機對準100元，就會顯示出AR物體。<br>
並且可以使用搖桿操控模型移動。
